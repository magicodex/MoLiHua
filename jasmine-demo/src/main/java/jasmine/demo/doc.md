## 代码风格

- 使用 UTF-8 编码。
- 使用 \n 换行符。
- 使用空格缩进。

## 目录结构

### module

```
|---- application
   |---- web
      |---- controller 控制器
      |---- dto
      |---- adapter 适配器
|---- business 业务层
   |---- service
   |---- domain 领域驱动对象
   |---- dto
   |---- adapter 适配器
|---- persistence 持久层
   |---- entity 数据实体
   |---- dao
   |---- param
   |---- mapper
|---- constant
|---- enums
|---- exception
|---- util
```
