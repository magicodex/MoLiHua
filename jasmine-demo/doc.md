## 代码风格

- 使用 UTF-8 编码。
- 使用 \n 换行符。
- 使用空格缩进。

## 目录结构

### module

```
|---- application
   |---- controller 控制器
   |---- dto
   |---- util
|---- business 业务层
   |---- service
   |---- dto
   |---- util
   |---- domain 领域驱动对象
|---- persistence 持久层
   |---- entity 数据实体
   |---- dao
   |---- dto
   |---- mapper
|---- constant
|---- enums
```
